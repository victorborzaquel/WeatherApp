# Weather App

## React Native App

Fiz um aplicativo para você conseguir ver como está o tempo em todas as cidades do mundo.  

Utilizei a api AccuWeather para pegar a previsão do tempo.

## Tecnologias

- React Native
- Styled Components
- Typescript
- Vector Icons
- React Navigation
- Inline Dotenv
- React Native svg
- API AccuWeather
